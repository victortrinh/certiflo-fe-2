## Getting Started

### Prerequisites

- yarn

  ```sh
  npm install --global yarn
  ```

### Installation

1. Install dependencies

   ```sh
   yarn
   ```

2. Change environment backend url to the proper one (default: http://localhost:5001) in env.ts

### Run application

```sh
yarn start
```
